13. l Parsing as Search
finding the right path through a finite-state automaton or finding the right transduction for an input can be viewed as a search problem.
In syntactic parsing, the parser can be viewed as searching through the space of possible parse trees to find the correct parse tree for a given sentence.so the search space of possible parse trees is defined by a grammar.
